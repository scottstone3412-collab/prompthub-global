Best ChatGPT Prompts for YouTube Script Writing

Copy, customize, and use these prompts to create engaging YouTube videos faster.

Prompt 1: Viral Storytelling Script

Prompt:

Write a 5-minute YouTube script about [topic] using a strong emotional hook, storytelling structure, suspense, and a clear call to action.

What it does

Creates a full storytelling-style script designed to keep viewers watching.

Best use case

Perfect for documentary, motivational, and African folktale channels.

Example result

You’ll get an intro hook, narrative flow, emotional tension, and an ending CTA.

Prompt 2: Educational YouTube Script

Prompt:

Create a beginner-friendly YouTube script explaining [topic] in simple language with examples and a short summary at the end.

What it does

Turns complex topics into easy-to-understand video scripts.

Best use case

Great for tutorial, finance, and student-focused channels.

Example result

You’ll receive a structured script with introduction, explanation, examples, and recap.

Prompt 3: Short-Form Video Script

Prompt:

Write a 60-second YouTube Shorts script about [topic] with a fast hook, concise explanation, and a memorable ending.

What it does

Generates quick, high-retention scripts optimized for Shorts.

Best use case

Ideal for viral facts, tips, and motivational content.

Example result

You’ll get a punchy script with short sentences and strong pacing.
