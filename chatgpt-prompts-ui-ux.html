ChatGPT Prompts for UI/UX Layout Concepts

Design intuitive, high-converting digital spaces. These expert prompts help you plan high-level web architecture, sketch responsive landing page wireframes, and map out seamless user user flows.

Prompt 1: The Landing Page Wireframe Architect

What it does: Structurally maps out a high-converting landing page layout section-by-section, indicating exactly where text, visual assets, and CTAs should sit for maximum impact.

Act as an expert UI/UX designer and conversion rate optimizer (CRO). I need to design a landing page wireframe for a website that offers [Insert Service/Product]. Provide a detailed text-based, section-by-section layout blueprint from top to bottom (Hero section down to the Footer). For each section, outline: 1) The visual layout structural priority, 2) The key copy/heading purpose, and 3) The exact placement and style of the interactive elements or Call-to-Action buttons.

Prompt 2: The User Flow & Interaction Map

What it does: Traces the exact path a user takes through an app or website, reducing friction and ensuring a smooth, logical interactive experience.

Act as a UX research specialist. Map out a step-by-step user interaction flow chart for an application feature where a user wants to [Insert Action, e.g., Register a business account or book a visual design consultation]. List out every action, decision pop-up window, and success screen needed to keep the user experience friction-free, quick, and highly intuitive.
