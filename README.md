# 🚀 PromptHub Global

Welcome to **PromptHub Global**, a premium, free open-source AI prompt library built to help content creators, entrepreneurs, copywriters, and digital designers unlock the absolute maximum potential of generative AI models like ChatGPT, Midjourney, and more.

This entire platform is built with lightweight, super-fast, and semantic HTML5 combined with a modern custom CSS3 dark-mode user interface.

---

## 📁 Repository Core Structure

Here is a map of how the library architecture connects:

* `index.html` — The main portal landing page.
* `writing-prompts.html` — Hub for creative, storytelling, and book authoring frameworks.
* `youtube-prompts.html` — Hub for video scripts, viral click-through titles, and SEO tagging tools.
* `business-prompts.html` — Hub for lean startup modeling, brand strategies, and social content calendars.
* `design-prompts.html` — Hub for Midjourney cinematic art formulas, logo ideation, and UI/UX mockups.
* `style.css` — Core responsive design stylesheet controlling global dark-mode aesthetics, layouts, and button interaction mechanics.

---

## ⚡ Key Technical Features

* **Zero-Dependency Deployment:** Built using pure vanilla code to guarantee instantaneous load speeds and pristine mobile layouts.
* **1-Click Clipboard Engine:** Integrated native JavaScript event listeners on every prompt container to allow users to copy prompt templates seamlessly to their clipboards with a single tap.
* **Modern Glass-morphic UI:** A beautifully tuned interface that prioritizes clean visual hierarchy, readability, and engaging subtle glowing animations.
